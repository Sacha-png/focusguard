Comment ça marche

Simple, radical

En 3 étapes, transforme ta relation aux écrans.

01

L'IA analyse ton planning

FocusGuard connecte ton Google Calendar et envoie tes événements à GPT-4o. L'IA calcule un quota de divertissement raisonnable selon ta charge de travail du jour.

Exam dans 8h · urgence 0.92
Réunion dans 2j · urgence 0.45
→ Budget conseillé : 45 min
02

Tu négocies (si besoin)

Pas d'accord avec la proposition ? Ouvre le chat et argumente : "j'ai travaillé 5h", "c'est le week-end"... L'IA est bienveillante et peut accorder 15-30 min supplémentaires.

Je te propose 45 min aujourd'hui. Tu as un examen demain matin.
ACCORD: 45 minutes
J'ai déjà travaillé 6h, je mérite un peu plus
Tu as raison, bon courage !
ACCORD: 75 minutes
03

Tu fais tes pompes et tu profites

Pour débloquer chaque session, complète le challenge pompes. Une fois validé, un compte à rebours se lance et tes sites favoris sont surveillés. Ultra-dépassement = popup.

Cloner et lancer avec pip
Installation manuelle (source)
# 1. Cloner le projet
git clone https://github.com/votre-user/focusguard.git
cd focusguard

# 2. Créer l'environnement virtuel
python -m venv .venv
.venv\Scripts\activate

# 3. Installer les dépendances
pip install -r requirements.txt

# 4. Lancer
python -m app.settings.dashboard_ui
